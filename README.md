WhatsFlow is a WhatsApp chatbot platform with CI/CD automation, built and managed entirely from an Android environment.

## Features
- WhatsApp Cloud API integration
- FastAPI backend
- GitHub Actions CI/CD
- Dockerized deployment
- Secure webhook handling
- Android (UserLAnd) friendly workflow

## Tech Stack
- Python (FastAPI)
- GitHub Actions
- Docker
- WhatsApp Cloud API

## Goal
Build, deploy, and scale a production-ready WhatsApp bot using only a mobile device.
